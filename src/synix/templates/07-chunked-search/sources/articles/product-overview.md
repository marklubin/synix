Acme CloudSync is a file synchronization platform designed for distributed teams. It provides real-time collaboration on documents, spreadsheets, and presentations across all major operating systems.

The platform uses a conflict-free replicated data type (CRDT) architecture to handle concurrent edits without requiring a central coordination server. This means team members can work offline and changes merge automatically when connectivity is restored.

CloudSync supports end-to-end encryption for all stored files. Encryption keys are derived from the user's passphrase and never leave the client device. Enterprise customers can integrate with their existing key management infrastructure via the KMS bridge API.

The search feature indexes document content in real time using an inverted index with BM25 ranking. Users can search across all synced folders, filter by file type or modification date, and preview results with highlighted matches. Search queries support boolean operators and phrase matching.

CloudSync pricing is based on storage tiers. The free tier includes 5 GB of storage and up to 3 team members. The Professional tier ($12/user/month) adds 100 GB, version history, and audit logs. The Enterprise tier includes unlimited storage, SSO integration, and a dedicated support channel.
