# NovaSight Analytics — Platform Overview

**Version: 4.2 | Updated: January 2026**

## Company Snapshot

NovaSight Analytics was founded in 2018 and is headquartered in Seattle, WA. The company has 280 employees, with 120 in engineering. Total funding to date: $85M (Series B closed June 2024). Current ARR is approximately $52M with 35% year-over-year growth. We serve 850+ customers across 14 countries.

## Platform Summary

NovaSight is an all-in-one analytics platform that combines real-time dashboards, custom ML model deployment, and enterprise-grade security in a single product. Our architecture is cloud-native and multi-tenant, running on AWS with optional hybrid deployment for regulated industries.

## Pricing Tiers

- **Starter**: $25/seat/month — core dashboards, 10 data connectors, email support, 5 GB storage per user
- **Professional**: $55/seat/month — custom ML models, REST API access, SSO/SAML, priority support, 25 GB storage per user
- **Enterprise**: custom pricing — hybrid deployment, dedicated instance, 24/7 support, custom SLAs, unlimited storage

Annual billing receives a 15% discount. Volume discounts available for 500+ seats.

## Core Features

- **Real-time dashboards**: sub-second query performance on datasets up to 10B rows
- **Custom ML model deployment**: bring your own Python or R models, deploy with one click
- **REST API**: 200+ endpoints covering all platform functionality, OAuth 2.0 authentication
- **Data connectors**: 85 native integrations including Salesforce, Snowflake, BigQuery, Redshift, PostgreSQL, MySQL
- **Collaboration**: shared dashboards, scheduled reports, annotations, and @mentions
- **Alerts**: threshold-based and AI-powered anomaly detection alerts via email, Slack, Teams, PagerDuty
- **Embedded analytics**: white-label dashboards embeddable in customer-facing applications

## Security & Compliance
